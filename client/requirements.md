## Packages
framer-motion | Smooth page transitions and scroll animations
lucide-react | Beautiful icons for the interface
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes efficiently
react-hook-form | Form handling
zod | Schema validation
@hookform/resolvers | Zod resolver for react-hook-form

## Notes
- Using Google Fonts: 'Outfit' for headings and 'Inter' for body text.
- Glassmorphism effects will be achieved with `backdrop-blur` and semi-transparent backgrounds.
- Data fetching uses `useQuery` and `useMutation` from `@tanstack/react-query`.
- Images will be sourced from Unsplash for placeholders.
